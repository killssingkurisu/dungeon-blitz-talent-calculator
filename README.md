# Dungeon Blitz Talent Calculator

Static GitHub Pages build of a Dungeon Blitz talent calculator.

## Disciplines

- Rogue / Viperblade
- Rogue / Shadowstalker

The calculator uses recovered game UI assets and local JavaScript data files, so it runs as a plain static website with no backend.

## Development

Open `index.html` directly, or serve the folder with any static file server.
